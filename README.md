## 💡主要功能

1. 支持为段落打**标签**
2. 支持快捷键**快速搜索输入**
3. 支持**导出数据库**为CSV文件
10. 支持**最小化**到**托盘**
11. 支持**跟随系统启动**
12. ......

## 💻支持平台（Qt 6.4.2）

- **仅支持64位**操作系统
- Windows 10 21H2 (1809 or later)
- Windows 11 21H2 (1809 or later)
- 计划中：Ubuntu 20.04 or later
